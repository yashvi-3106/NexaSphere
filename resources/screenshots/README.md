## Contributing Screenshots

When adding new screenshots:

1. **Naming convention**: `{page/feature}-{variant}.png`  
   Examples: `events-page.png`, `homepage-dark.png`, `admin-login.png`

2. **Dimensions**: Capture at 1440×900 for desktop, 375×812 for mobile

3. **Format**: PNG (preferred) or WebP for smaller file sizes

4. **Quality**: Use a clean browser window with realistic (non-lorem) content

5. **Dark mode**: Prefer dark mode screenshots where the UI supports it
