## Contributing Resources

When adding new resources:

- **Brand files** → `brand/` (SVG/PNG logos, `.json` color palettes, font names)
- **App screenshots** → `screenshots/` (use descriptive names like `homepage-dark.png`, `admin-events.png`)
- **Design mockups / wireframes** → `screenshots/` or a dedicated `mockups/` subfolder
- Keep image file sizes reasonable — prefer optimised PNGs or WebP
